How to use

<input type="file" ng-model="file" class="form-control" ng-fileupload="myFile" id="file" name="file" file-error="fileError1" fileSize="53600" fileType='"image/png","image/jpeg","image/jpg"' multiple>

Attributes

  • ng-fileupload : pass the scope variable for access anywhere within projects (eg. ng-fileupload="files" )
  • fileType :Define file type
    $scope.filetype=["image/png","image/jpeg","image/jpg"];
    fileType="{ {filetype} }"

    OR

    fileType='"image/png","image/jpeg","image/jpg"'
  • fileSize :Define file size in bytes
    $scope.fileSize=523020;
    fileSize="{ {fileSize} }"

    OR

    fileSize="523020"
  • file-error :Define error

Function/method

Upload Mulitiple file

fileUpload.uploadFileToUrl(files, uploadUrl,callback);
files: scope variable

Examples:
$scope.files={};

uploadUrl: server url where you send file

Examples:
$scope.uploadUrl="http://example.com/getFiles";

callback: function for recived response form server

Examples:
fileUpload.uploadFileToUrl(files, uploadUrl,function(){
$log.info(response)
});

Remove File

$scope.removeFile=function(index){ fileUpload.remove($scope.files,index); }

Upload single file

$scope.uploadSignle=function(index){ var status=fileUpload.uploadSignle($scope.files, index, uploadUrl ,function(response){ $log.info(response); }); }

index: index value where you want to upload for FileList
fileList=[file1,file2, file3] if index=2;
